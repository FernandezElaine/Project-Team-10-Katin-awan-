# Katin-awan: AI-Enabled Transparency Portal

## Project Overview
Katin-awan is a web-based transparency portal designed to improve barangay governance by making budget, expense, and project information more accessible to residents. It integrates AI, OCR-based document processing, and geospatial tracking to promote accountability and citizen participation.

## Purpose
The system addresses the lack of transparency in barangay financial reporting by providing a centralized platform where residents can view budgets, track projects, and submit feedback in real time.

## Key Features
- User registration and role-based access (Resident / Admin)
- Budget tracking dashboard with visual charts
- Project monitoring with geospatial mapping
- OCR-based expense document analysis
- AI validation for financial records
- Multilingual chatbot (English, Tagalog, Bisaya)
- Anonymous feedback system
- Audit logs for accountability

## Tech Stack
- HTML, CSS, JavaScript
- Supabase (Database + Auth)
- OCR API
- Google Gemini AI API
- Leaflet.js (Maps)

## Project Structure
- `/js` – JavaScript files
- `/css` – Stylesheets
- `/pages` – HTML pages
- `/data` – JSON data files
- `/assets` – images and documents
